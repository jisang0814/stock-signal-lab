# Stock Signal Lab 진행상황 (2026-02-14)

## 1) 프로젝트 목적
- 국내/미국 주식 대상 시그널 분석 및 리스크 관리 대시보드
- 실거래 주문은 하지 않고, 분석/추천/리스크 통제/알림 중심 운용

## 2) 현재 구현 완료 기능

### 핵심 분석
- 멀티페이지 Streamlit 앱 구성
- 시그널 엔진(추가매수/보유/매도, 점수/신뢰도, 시장국면)
- 지표: RSI, MACD, 모멘텀, 변동성, ATR, 이동평균 기울기 등
- 백테스트/워크포워드 품질 검증
- 벤치마크 대비 알파/MDD/Sharpe Edge 비교

### 포지션/품질
- 포지션 손익 계산(평단/수량 입력)
- 리스크 기반 권장 포지션 사이징
- 신호 품질 교정(Calibration):
  - 구간별 실제 승률/평균수익
  - 기대값/브라이어 점수(Brier)
- 품질 기반 포지션 배수 자동조정(공격/균형/보수 프리셋 반영)

### 운영/리스크
- Execution Simulator (수수료/슬리피지/부분체결)
- Rebalance 자동 계획(임계치/비용 반영)
- Risk Kill-Switch (변동성/MDD/이벤트 리스크)
- Drift Monitor (최근 성능 붕괴 감지)
- Portfolio Risk (상관관계/집중도/리스크 기여/가중치 제안)

### 로깅/추천
- Signal Log 저장/필터/성과 라벨링
- 자동 프리셋 추천 및 추천 이력 추적(Reco Track)

### 알림
- Telegram 연동 완료
- 테스트 발송 성공 확인
- Kill-Switch/Drift 자동 알림 + 쿨다운 중복방지
- 알림 이력(sent/failed/skipped) 저장/조회/초기화
- Dashboard 최근 알림 상태 카드 + 최근 5건 타임라인

### 운영 체크
- Ops Checklist 페이지 추가
- Dashboard에 Checklist 미완료 경고 배너 + 이동 링크
