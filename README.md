# Value-style Stock Lab

실매매 의사결정에 활용할 수 있도록, 시그널 품질/리스크 관리를 강화한 멀티페이지 주식 분석 앱입니다.

## 페이지

- `📊 Dashboard`
  - 시장 히트맵(변동률 색상, 시총/거래대금 크기)
  - Top 5 랭킹(시가총액/거래대금/점수)
  - AI 스타일 분석 카드
  - 최근 알림 상태(sent/failed/skipped) 요약
  - 최근 5건 알림 타임라인
  - Ops Checklist 미완료 항목 경고 배너
- `🔎 Screener`
  - 점수/시그널/RSI 필터 스크리너
  - 최소 거래대금(유동성) 필터
- `🧠 Analysis`
  - 개별 종목 시그널/지표 분석
  - 포지션 손익 계산(수수료 무시)
  - 일봉/주봉 다중 타임프레임 합의 체크
  - 벤치마크 대비 알파/드로우다운 비교
  - 리스크 기반 권장 포지션 사이징
  - 워크포워드 품질 검증
  - 전략 프리셋(보수/균형/공격)
  - 최근 90일 로그 기반 자동 프리셋 적용(시장별, 최소 표본수 조건)
  - 실적발표 이벤트 리스크 반영
  - 사용자 정의 거시 이벤트 캘린더 리스크 반영
  - 이벤트 캘린더 CSV 업로드/다운로드
  - 이벤트 전후 성과 회고(자산 vs 벤치마크)
  - 섹터별 이벤트 회고(평균 알파)
  - 신호 품질 기반 포지션 크기 조정 배수 제안
- `📒 Signal Log`
  - 저장한 시그널 로그
  - 기간/시장/프리셋/점수 필터
  - 최근 로그 추적 성과 요약
  - 슬리피지 반영 성과
  - 프리셋(보수/균형/공격)별 성과 비교
  - N일 후 실제 성과 라벨링(HIT/MISS)
  - 최근 90일 기준 자동 프리셋 추천(최소 표본수 조건)
- `🗂 Reco Track`
  - 추천 프리셋 로그 이력 조회
  - 시장/기간 필터
  - 시간별 추천 점수 추이 및 분포
- `🧪 Execution Simulator`
  - 수수료/슬리피지/부분체결(ADV 제한) 기반 체결 시뮬레이션
  - 신호 품질(교정 기대값/승률) 기반 주문 크기 자동조정
- `⚖️ Rebalance`
  - 임계치/비용 반영 자동 리밸런싱 계획 생성
- `🚨 Risk Kill-Switch`
  - 변동성/MDD/이벤트 리스크 기반 Risk-Off 자동 판단
  - 조건 충족 시 Telegram 자동 알림(쿨다운 기반 중복 방지)
- `📉 Drift Monitor`
  - 최근 성과 붕괴(드리프트) 탐지
  - 드리프트 감지 시 Telegram 자동 알림(쿨다운 기반 중복 방지)
- `🔔 Alerts`
  - Telegram 알림 설정/테스트 발송
  - 알림 발송/스킵/실패 이력 조회 및 초기화
- `🧬 Signal Quality`
  - 신뢰도 구간별 실제 승률/평균 수익률 교정(캘리브레이션)
  - 기대값/브라이어 점수(Brier Score)로 신호 품질 정량화
- `✅ Ops Checklist`
  - Telegram/알림/로그 상태 운영 점검
  - 체크리스트 저장/전체 해제
- `🧩 Portfolio Risk`
  - 상관관계 히트맵
  - 포트폴리오 집중도(HHI), 유효 종목수
  - 종목별 리스크 기여도
  - 목표 변동성 기반 가중치 제안
  - Shrinkage 공분산(λ) 기반 안정화
  - 최대 종목 비중 제한(캡)
  - 제안 가중치 CSV 다운로드/복사 텍스트
